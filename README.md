# BookFlow - Biblioteca Virtual

BookFlow es un sistema de gestión de libros electrónicos desarrollado para la asignatura de Programación Orientada a Objetos II.

## Objetivo

Desarrollar una biblioteca virtual que permita organizar libros electrónicos, administrar usuarios y facilitar la búsqueda y el seguimiento de lecturas.

## Funcionalidades principales

- Registro e inicio de sesión de usuarios.
- Consulta del catálogo de libros.
- Búsqueda por título, autor o género.
- Gestión de libros favoritos.
- Registro del estado de lectura.
- Administración de libros y usuarios.

## Módulos del sistema

1. Gestión de usuarios y autenticación.
2. Catálogo de libros.
3. Lecturas y favoritos.
4. Administración.

## Tecnologías

- Lenguaje de programación Go.
- Archivos JSON para almacenar información.
- GitHub para el control de versiones.
